## Learning path

- New to TS: Get Started → Handbook (Basics, Everyday Types, Narrowing, Functions, Object Types).
- Need types from types: Handbook → Creating Types from Types (Generics, keyof, typeof, Conditional, Mapped, Template Literal).
- Config: tsconfig.json and TSConfig Reference.
- React/JSX: Reference → JSX.

## Relationship to JavaScript

TypeScript shares syntax and runtime with JavaScript. For “how do I do X in TypeScript?” (e.g. sort a list, fetch API), the answer is usually the same as in JavaScript; TypeScript adds types and compile-time checks. Use JS resources when the question is about runtime behavior.
